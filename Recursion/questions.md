# Recursion Practice Questions

## 1. Mathematical Problems
- **Factorial:** Write a recursive function to find the factorial of $n$.
- **Power Function ($x^n$):** Implement a recursive function to calculate $x$ raised to the power $n$.
- **Sum of Digits:** Write a recursive function to calculate the sum of digits of a number.

## 2. Fibonacci Special Focus
- **Recursive Solution:** Implement the simple recursive Fibonacci.
- **Iterative Solution:** Implement an efficient loop-based Fibonacci.
- **Optimized (DP):** Implement Fibonacci using a memoization array to store intermediate results.
- **Exam Variation 1:** Write a program to print the $n^{th}$ Fibonacci number.
- **Exam Variation 2:** Write a program to print the Fibonacci series up to $n$ terms.
- **Complexity Comparison:** Discuss the time and space complexity of all three Fibonacci versions.

## 3. Advanced Recursion
- **Tower of Hanoi:** Write a recursive function to solve the Tower of Hanoi problem for $n$ disks.
- **String Reversal:** Write a recursive function to reverse a string in-place.
- **Subset Sum:** (Optional) Find all subsets of a set that sum to $k$.

## 🔥 MOST IMPORTANT REPEATABLE QUESTIONS
- **Fibonacci (Recursive + Iterative + DP Comparison)**
- **Tower of Hanoi**
- **Factorial Calculation**
