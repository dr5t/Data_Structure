# Sorting and Searching Practice Questions

## 1. Searching
- **Linear Search:** Implement linear search for an array of $n$ elements.
- **Binary Search:** Implement binary search on a sorted array.
- **Interpolation Search:** (Optional) Implement interpolation search for uniformly distributed data.

## 2. Sorting
- **Bubble Sort:** Implement bubble sort with optimized early exit (if the array is already sorted).
- **Selection Sort:** Implement selection sort with $O(n^2)$ time.
- **Insertion Sort:** Implement insertion sort with $O(n^2)$ time.
- **Merge Sort:** Implement merge sort using recursion and divide-and-conquer.
- **Quick Sort:** Implement quick sort with a random pivot or last-element pivot.

## 3. Advanced Challenges
- **Sort K Sorted Array:** Find how to sort an array where each element is $k$ positions away from its sorted position.
- **Kth Smallest Element:** Find the $k^{th}$ smallest element using quick-select.
