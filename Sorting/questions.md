# Sorting and Searching Practice Questions

## 1. Searching

- **Linear Search:** Implement linear search for an array of $n$ elements.
- **Binary Search:** Implement binary search on a sorted array.
- **Interpolation Search:** (Optional) Implement interpolation search for uniformly distributed data.

## 2. Sorting

- **Bubble Sort:** Implement bubble sort with optimized early exit (if the array is already sorted).
- **Selection Sort:** Implement selection sort with $O(n^2)$ time.
- **Insertion Sort:** Implement insertion sort with $O(n^2)$ time.
- **Merge Sort:** Implement merge sort using recursion and divide-and-conquer.
- **Quick Sort:** Implement quick sort with a random pivot or last-element pivot.

- **Kth Smallest Element:** Find the $k^{th}$ smallest element using quick-select.

---

## 🌳 BEYOND SYLLABUS (MASTER SET)

### 1. Advanced Binary Search

- **First and Last Occurrence ⭐⭐**: Given a sorted array with duplicates, find the first and last position of a target.
- **Search in Rotated Sorted Array ⭐⭐⭐**: Implement an $O(\log n)$ search for an array that has been rotated.
- **Find Peak Element**: Find an element that is strictly greater than its neighbors in $O(\log n)$.

### 2. Expert Sorting Logic

- **QuickSelect ⭐⭐**: Find the K-th smallest element in $O(n)$ average time using the partitioning logic.
- **Merge Sort for Linked Lists**: Implement Merge Sort specifically for a Singly Linked List.
- **Counting Sort logic**: (Bonus) Implement or discuss when non-comparison sorting ($O(n)$) is used.

### 3. Optimization Challenges

- **Find the Square Root**: Use Binary Search to find the integer square root of `x` in $O(\log x)$.
- **Search in a 2D Matrix**: Search for a value in an $M \times N$ matrix where rows and columns are sorted.
