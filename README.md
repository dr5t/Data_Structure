## ⚡ Quick Complexity Reference

| Algorithm | Best Time | Worst Time | Space |
| :--- | :--- | :--- | :--- |
| **Binary Search** | $O(1)$ | $O(\log n)$ | $O(1)$ |
| **Bubble Sort** | $O(n)$ | $O(n^2)$ | $O(1)$ |
| **Insertion Sort** | $O(n)$ | $O(n^2)$ | $O(1)$ |
| **BST Search** | $O(\log n)$ | $O(n)$ (Skewed) | $O(h)$ |
